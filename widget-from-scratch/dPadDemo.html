The D-Pad

DPad will navigate table! Use the Up and Down Arrow Keys to Move From Row to Row
The selected row will be highlighted in grey!
The D-Pad will also highlight which key you press on your keyboard.
Click functionality not currently support on the D-Pad.
To use in code just build a table! When constructing your rows, set their id equal to the row number (ex. row 1 id = 1) and the row class to "tRow". Give whatever row you want to start focus on the class "vertical-target" as well. Last, set the table class to "dpad-receiver" and you should be good to go!
